# Personal Portfolio

Minimal personal website with links to GitHub, LinkedIn, and resume.

## Tech Stack

- Vue 3 + TypeScript
- Vite

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Structure

```
public/
  └── Landen_English_Resume.pdf   # Static resume file
src/
  ├── App.vue                     # Main component
  ├── style.css                   # Global styles
  └── main.ts                     # Entry point
```
